系统信息

获取系统名称和版本

检查系统版本位数

如果是64位系统则返回”AMD64”否则返回”X86”

查看系统所有环境变量

查看计算机所有用户

NET USER

查看指定用户详细详细

NET USER USERNAME

网络信息

查看基本网络情况

IPCONFIG /ALL

查看路由表

route print

查看ARP缓存

arp -A

防火墙相关

查看端口占用情况
netstat -ano

防火墙配置
netsh firewall show config

防火墙状态
netsh firewall show state

应用程序和服务

查看正在运行的程序和服务

schtasks /QUERY /fo LIST /v
以list的形式输入详细信息

查看服务的进程ID

tasklist /SVC

查看已安装的驱动程序列表

DRIVERQUERY

也可以通过WMIC获取更多信息

查看已安装程序相关信息(包括程序名、供应商、版本、识别号等)
Caption、Identifying Number 、Name、Vendor、Version
比如我安装的XShell相关信息如下

按照类似的方法还可以获取 服务、进程、和启动项相关信息

查看是否有可作为非特权用户执行的msi文件
如果没有则会返回”错误: 系统找不到指定的注册表项或值。”

查看setuid或setgid是否被修改过
只存在于Windows Servers中

查看系统更新记录
wmic qfe get Caption,Description,HotFixID,InstalledOn
